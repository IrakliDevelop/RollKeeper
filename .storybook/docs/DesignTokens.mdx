# Design Tokens

Design tokens are the foundational values of the RollKeeper design system. They ensure consistency across all components and make theming easier.

All tokens are defined in `src/components/ui/primitives/design-tokens.ts`.

## Spacing

Based on a 4px scale for consistent rhythm.

| Token | Value | Usage |
|-------|-------|-------|
| `xs` | 4px | Tight spacing, icon gaps |
| `sm` | 8px | Small gaps, padding |
| `md` | 12px | Default padding |
| `lg` | 16px | Section spacing |
| `xl` | 24px | Large spacing |
| `2xl` | 32px | Section gaps |
| `3xl` | 48px | Page sections |
| `4xl` | 64px | Hero spacing |

```typescript
import { spacing } from '@/components/ui/primitives';

// Usage
<div style={{ padding: spacing.md }}>Content</div>
```

## Typography

### Font Sizes

| Token | Size | Usage |
|-------|------|-------|
| `xs` | 12px | Captions, badges |
| `sm` | 14px | Body text, inputs |
| `base` | 16px | Default body |
| `lg` | 18px | Large body |
| `xl` | 20px | Subheadings |
| `2xl` | 24px | Section headings |
| `3xl` | 30px | Page titles |
| `4xl` | 36px | Hero text |

### Font Weights

| Token | Weight | Usage |
|-------|--------|-------|
| `normal` | 400 | Body text |
| `medium` | 500 | Labels, emphasis |
| `semibold` | 600 | Subheadings |
| `bold` | 700 | Headings |

## Shadows

Used for elevation and depth.

| Token | Usage |
|-------|-------|
| `sm` | Subtle shadow for cards |
| `md` | Default elevation |
| `lg` | Prominent shadow for modals |
| `xl` | Large shadow for overlays |
| `inner` | Inset shadow for inputs |

---

## Border Radius

| Token | Value | Usage |
|-------|-------|-------|
| `sm` | 6px | Small elements |

| `lg` | 12px | Cards |
| `xl` | 16px | Large containers |
| `2xl` | 24px | Pills, avatars |
| `full` | 9999px | Circles |

## CSS Variables (Theming)

The design system uses CSS custom properties for light/dark mode theming.

### Light Mode (default)

```css
:root {
  --background: #ffffff;
  --foreground: #171717;
  --color-bg-primary: #ffffff;
  --color-bg-secondary: #f8fafc;
  --color-text-primary: #1e293b;
  --color-text-secondary: #475569;
  --color-border-primary: #e2e8f0;
}
```

### Dark Mode

Applied with `[data-theme="dark"]` on the root element.

```css
[data-theme="dark"] {
  --background: #0a0a0a;
  --foreground: #ededed;
  --color-bg-primary: #0f172a;
  --color-bg-secondary: #1e293b;
  --color-text-primary: #f1f5f9;
  --color-text-secondary: #cbd5e1;
  --color-border-primary: #334155;
}
```

Use the theme toggle in the Storybook toolbar to preview components in both modes.

## Usage in Components

```typescript
import { 
  colors, 
  spacing, 
  typography, 
  shadows, 
  borderRadius 
} from '@/components/ui/primitives';

// Direct usage
const styles = {
  color: colors.primary[600],
  padding: spacing.lg,
  fontSize: typography.fontSize.sm,
  boxShadow: shadows.md,
  borderRadius: borderRadius.lg,
};
```

For most cases, prefer using Tailwind CSS classes which align with these tokens.
